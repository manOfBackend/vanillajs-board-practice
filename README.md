## 프로젝트 구조
- src
    - components
        - 데이터 출력 역할 (html 포함)
    - constants
        - 개발에 필요한 상수들
    - modules
        - 개발에 필요한 모듈 (리액트, 라우터 등)
    - pages
        - 데이터 처리 역할 (fetch)
    - services
        - api 처리
    - utils
        - 개발에 필요한 유틸들 (로컬 스토리지, 시간 처리 등)
    - App.js
    - App.css
    - index.js

---
## 실행 방법
### `npm i`
### `npm start`: development mode
### `npm test`: testing
### `/dashboard`: 대시보드
### `/pages/:pageNum`: 대시보드 페이지
